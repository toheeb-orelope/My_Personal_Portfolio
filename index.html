  • Snack Man Game An interactive web game built using HTML, CSS, & JavaScript

    In my first year, second semester at the University of Northampton, I took a web development module where we were tasked with building an interactive web game called Snack Man using HTML, CSS, and JavaScript. In this game, users could move the player around to collect points while avoiding collisions with enemies. The game featured a leaderboard that saved the highest scores, and players could control the game using on-screen buttons or their keyboard, with a live score display. Upon colliding with enemies, a life will be deducted from the initial three, and if the lives reached zero, the player would prompted to enter their name for the leaderboard, with scores saved using local storage. The development process went smoothly, thanks to our dedicated lecturer, who delivered engaging lectures and provided exceptional support to all students. His commitment to our success was commendable, and I felt privileged to learn from such an inspiring mentor.

    Key Game Features:

    • Player Movement: Users can control the main character, navigating the game to collect points while avoiding enemies.
    • Collision Detection: The game features a system to detect collisions with enemies, deducting lives from the player. Each player starts with three lives, and upon losing all lives, a prompt appears to enter their name for the leaderboard.
    • Leaderboard: High scores are saved on a leaderboard using local storage, allowing players to see their ranking.
    • Control Options: Players can control the game using on-screen buttons or their keyboard, with a score display that updates live as they play.

    This project will enhance students' understanding of HTML, CSS, and JavaScript implementation.
